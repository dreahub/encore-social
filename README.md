# Encore

Encore is a low-pressure, private-first social media platform designed for intentional sharing with friends.

## Core Ideas
- Hidden like counts
- Hidden follow counts
- Profiles private by default
- Friend-only sharing
- Required content tags on every post
- Calm, anxiety-reducing design

## Post Types
1. Photo Set
2. Mini Blog
3. Mini Vlog

## Design Direction
- Pale beige base
- Soft pastel accents
- Rounded UI
- No aggressive notifications
- Cozy, calm, low-pressure feel

## Current Status
Early concept / planning stage.
